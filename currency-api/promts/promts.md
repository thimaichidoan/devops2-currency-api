# Prompts utilisés pendant le développement

## Prompt 1

Créer un projet Spring Boot avec Maven en Java permettant de développer une API REST.

---

## Prompt 2

Créer une architecture propre avec les packages :

- controller
- service
- client
- dto

---

## Prompt 3

Créer une API REST permettant de convertir un montant d'une devise vers une autre.

---

## Prompt 4

Créer un service Java utilisant WebClient afin d'appeler une API REST externe de taux de change.

---

## Prompt 5

Créer un contrôleur REST POST `/api/currency/convert` recevant un JSON et retournant le résultat de la conversion.

---

## Prompt 6

Expliquer les erreurs Maven, Java et Spring Boot rencontrées pendant le développement et proposer leur correction.
